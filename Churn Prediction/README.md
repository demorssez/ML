Предсказание оттока клиентов

https://www.kaggle.com/datasets/blastchar/telco-customer-churn 

Проанализировал датасет, применил standardscaler для числовых и onehotencoder для категориальных признаков. Сравнил logisticregression, randomforest, svm и xgboost. На разных моделях провел gridsearchcv и среди них выбрал модель с наилучшими результатами по метрикам. Дальше провел визуализацию по результатам этой модели.

По метрикам на тесте лучшая модель это XGBoost с Recall 0.83 и Precision 0.55. Выбрал recall потому что пропустить уходящего клиента выйдет дороже чем пробовать пытаться переубедить его

Важнее всего признаки план контракта и тип подключения к интернету. Модель позволяет заранее выявить 83% клиентов склонных к оттоку
